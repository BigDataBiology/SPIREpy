SPIREpy
=======

Welcome
-------


SPIREpy is a Python package and command-line tool that allows users to interact
with the SPIRE_ database in a more convinient way.

.. _SPIRE: https://spire.embl.de/

Contents
--------

.. toctree::
   :maxdepth: 1

   Quickstart <quickstart>


Installation
------------

SPIREpy is available on PyPI and can be installed via

.. code-block:: bash

   pip install spirepy
